本主题已在 `astro.config.mjs` 中安装并配置 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成。若不需要 MDX，可从配置里移除该集成以关闭支持。

## 为什么用 MDX？

MDX 是 Markdown 的扩展，支持内嵌 JavaScript 与 JSX。这样可以把 [JavaScript 与 UI 组件写进 Markdown 内容](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro)，例如交互图表或提示框。

若你已有 MDX 内容，该集成通常能让迁移到 Astro 更顺畅。

下面演示如何在 MDX 里导入并使用 UI 组件。

	MDX 内嵌组件
</HeaderLink>
